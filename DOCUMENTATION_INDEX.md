## üöÄ Quick Reference

### Essential Commands

```bash
# Start Backend
cd backend && npm run server

# Start Frontend
cd frontend && npm run dev

# Start Admin Panel
cd Admin && npm run dev

# Start Stall Admin
cd StallAdmin && npm run dev
```

### Important URLs

- **Frontend**: http://localhost:5173
- **Backend API**: http://localhost:4000
- **Admin Panel**: http://localhost:5174
- **Stall Admin**: http://localhost:5175

### Key Ports

- Backend: `4000`
- Frontend: `5173`
- Admin: `5174`
- Stall Admin: `5175`
